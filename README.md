# PromptHub

واجهة موقع Prompt Gallery مبنية بـ React + Vite.

## التشغيل محليًا

```bash
npm install
npm run dev
```

## البناء للنشر

```bash
npm run build
```

## النشر على Vercel

1. ارفع الملفات إلى GitHub.
2. من Vercel اختر Import Project.
3. اختر هذا المستودع.
4. اضغط Deploy.

Vercel سيتعرف على Vite تلقائيًا.
